# Typecho-Theme-AirCloud
一个Typecho主题

移植自[hexo-theme-aircloud](https://github.com/aircloud/hexo-theme-aircloud)

### 主题功能
1. 自动解析 h1 h2 h3 等标签，生成 toc 目录树

2. 使用 highlight.js 实现代码高亮

3. 可以自定义首页头像下的用户名，默认是管理员账号的昵称

4. 自定义首页头像

5. 头像方/圆形切换

6. 文章首行自动缩进

7. 备案号设置
